## 🏙️ Airbnb NYC EDA — Exploratory Data Analysis

A complete Exploratory Data Analysis (EDA) project built using Python and Jupyter Notebook, focused on analyzing Airbnb listings in New York City to uncover pricing trends, neighborhood patterns, room type distributions, availability insights, and host behavior through statistical analysis and visualizations.

## 🚀 Features
* **Comprehensive Airbnb EDA** – Analyze NYC Airbnb listings and market trends.
* **Neighborhood Analysis** – Compare listings across different NYC boroughs.
* **Price Distribution Insights** – Explore pricing patterns and outliers.
* **Room Type Analysis** – Analyze private rooms, entire homes, and shared rooms.
* **Availability Analysis** – Study listing availability throughout the year.
* **Host Activity Insights** – Understand host behavior and listing frequency.
* **Interactive Data Visualizations** – Generate charts and plots for business insights.
* **Correlation & Statistical Analysis** – Discover relationships between listing features.
* **Data Cleaning & Preprocessing** – Handle missing values and inconsistent records.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/Univariate_and_Bivariate_Analysis.git
```
2. **Navigate to the project folder**:
```
cd Univariate_and_Bivariate_Analysis
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
Airbnb_NYC_EDA.ipynb
```
